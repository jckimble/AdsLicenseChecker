Provides the classes to make life alittle easier with android market licensing.
This android library is open source "Fork me at GitHub". This online javadoc is also packaged for offline use here. The pre-compiled library is here.

This library was created by James C Kimble Jr

To build this library you will need:
android-sdk
google play licensing
admob-sdk

You can build this android library yourself by running the following:
git clone git://github.com/jckimble/AdsLicenseChecker.git
cd AdsLicenseChecker
ant dist

To cleanup workspace run:
ant cleanup
